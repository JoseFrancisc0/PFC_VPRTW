# PFC_VRPTW

Comandos para su funcionamiento:
* Ejecutar cmake --build . --config Release para compilar el proyecto
* Probar con .\build\Release\ALNS_VRPTW.exe
* Ejecutar automate.py para acumular los resultados
* Ejecutar evaluate_models.py para comparar resultados con el benchmark
* Ejecutar generate_boxplots.py para crear los gráficos de boxplots
